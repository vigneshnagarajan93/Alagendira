# Alagendira Exports

A premium, motion-heavy digital platform for Alagendira Exports, a global export house specializing in high-quality garments and textile solutions. Built with a focus on immersive storytelling, modern animations, and a seamless B2B user experience.

## 🚀 Project Overview

This website is designed to showcase Alagendira Exports' heritage, manufacturing excellence, and commitment to sustainability. The design philosophy follows a "Modern Organic" aesthetic with high-end scrollytelling features.

### Key Features:
- **Immersive 4D Interactions**: Velocity-based text unmasking and magnetic component effects.
- **Scrollytelling**: Engaging scroll-triggered animations to narrate the brand's journey.
- **Premium Design**: Apple-inspired clean aesthetic with refined typography and harmonious color palettes.
- **B2B Optimized**: Clear categorization of exports and simplified inquiry flows.

## 🛠️ Local Development

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Setup Instructions
1. **Clone the repository**:
   ```bash
   git clone [repository-url]
   cd Alagendira
   ```

2. **Navigate to the frontend directory**:
   ```bash
   cd frontend
   ```

3. **Install dependencies**:
   ```bash
   npm install
   ```

4. **Start the development server**:
   ```bash
   npm run dev
   ```

The application will be available at `http://localhost:5173`.

## 🎨 Design System

- **Colors**: Sage, Linen, Slate (Modern Organic palette)
- **Typography**: Apple-inspired sans-serif (Inter/SF Pro)
- **Frameworks**: Framer Motion for animations, Tailwind CSS for styling.
